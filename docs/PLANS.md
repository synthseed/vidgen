# PLANS
Owner: platform
Status: active
Last Reviewed: 2026-02-23

Execution plans are mandatory for any non-trivial change.

## Plan Lifecycle
1. Create a plan in `exec-plans/active/` with scope, constraints, risks, and checkpoints.
2. Update progress in-place during implementation.
3. When complete, move to `exec-plans/completed/` with outcomes and follow-up debt.
4. Record debt in `exec-plans/tech-debt-tracker.md`.

## Plan Template
- Context
- Objective
- Constraints
- Tasks
- Validation
- Risks
- Outcomes
